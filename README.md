nullpointer-i18n - RequireJS-модуль интернационализации
=======================================================

Окружение
---------

Установить:

* node.js 0.10.x+
* npm 1.3.x+
* grunt-cli <code>npm install grunt-cli -g</code>
* bower <code>npm install bower -g</code>

В корне проекта выполнить <code>npm install</code>

Сборка
------

    grunt
    ? grunt package


Запуск тестов
-------------

    grunt test
